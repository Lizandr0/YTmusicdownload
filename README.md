# YTMusicDownload

Aplicación para descargar música en formato **MP3** de forma rápida y sencilla desde la terminal, principalmente en Linux, Temux y/o consola de windows.

---
- Formato MP3
- Linux
- Temux
- Windows
  
## Instalación

```bash
git clone https://github.com/Lizandr0/YTmusicdownload
cd YTmusicdownload
pip install -r requirements.txt
python ytmusic.py
```
## ATENCION

Puede que necesites configurar tu carpeta de archivos, cambialo en la linea 72:

'outtmpl': '/aqui va tu carpeta/%(title)s.%(ext)s'
